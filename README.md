#  Supabase Task Manager

Un'applicazione Full-Stack moderna per la gestione dei task, con architettura containerizzata tramite **Docker** e persistenza dei dati sul cloud di **Supabase**.

##  Funzionalità
- **Create**: Aggiunta di nuovi task in tempo reale.
- **Read**: Visualizzazione dinamica dei task salvati nel database.
- **Update**: Possibilità di sbarrare i task completati con un checkbox.
- **Delete**: Rimozione definitiva dei task tramite l'icona del cestino.

## 🛠️ Stack Tecnologico
- **Frontend**: HTML5, JavaScript (Fetch API), Tailwind CSS (tramite CDN)
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL ospitato su **Supabase**
- **DevOps**: Docker, Docker Compose
